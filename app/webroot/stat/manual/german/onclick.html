Klick-Zähler


Die PHP Web Stat bietet auch die Möglichkeit, Klicks auf Links/Dateien aufzuzeichnen. Um diese Funktion zu nutzen, bedarf es eines zusätzlichen Eingriffs in die aufzuzeichnenede Webseite. Kopieren Sie für diese Erweiterung, wie schon unter Schritt 4 beschrieben, zusätzlich das folgende Javascript in jede aufzuzeichnende Seite zwischen <body> und </body>:

<script type="text/javascript" src="http://www.example.com/stat/track_file.php"></script>

Bitte achten Sie auch hier darauf, dass der vollständige Pfad zu der auf dem Server liegenden TRACK_FILE.PHP angegeben werden muss!

Anschliessend ist für jeden Klick, der aufgezeichnet werden soll, ein Javascript-Code in den HTML-Tag des jeweiligen Objektes einzubauen. Der Befehl lautet in Rohfassung:

onfocus="track_file('Beispiel_ID')"

Wenn Sie also beispielsweise den Klick auf einen Link aufzeichnen möchten, so wäre dieses Beispiel passend für Sie:

Bisher:         <a href="link.html">Link</a>
Angepasst:  <a href="link.html" onfocus="track_file('Beispiel_ID')">Link</a>

Bitte ersetzen Sie den Wert für Beispiel_ID durch einen von Ihnen gewählten Wert.


Die Auswertung der aufgezeichneten Klicks finden Sie dann innerhalb Ihrer Statistik in den Plugins unter dem Namen Klick-Zähler.
