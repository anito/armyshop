Click Counter


The PHP Web Stat offers a click tracker that enables you to track clicks on links. If you want to use this function, you have to be aware of how to edit the HTML code of your website. First of all, include (see manual step 4) the following code of JavaScript within the BODY tags of your website:

<script type="text/javascript" src="http://www.example.com/stat/track_file.php"></script>

Please be aware, that the complete external path to the stat file TRACK_FILE.PHP has to be edited to your needs!

Afterwards every object (link) its clicks should be tracked, needs its own JavaScript code within its HTML tag:

onfocus="track_file('Example_ID')"

The following example gives you a howto of embedding the code:

Original:   <a href="link.html">Link</a>
Embedded:   <a href="link.html" onfocus="track_file('Example_ID')">Link</a>

Please replace Example_ID with a desired name that fits your needs.


The report of the clicked links can be found within the Plugin section of the PHP Web Stat, named under Click Counter.
